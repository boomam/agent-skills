# agent-skills
Self-created agent skills for usage with various AI Agents.  

## Tested with
Nanobot.  

## Current Skills
### Pangolin
#### Pre-Reqs
* Functional Pangolin Deployment  
* [Enabled API](https://docs.pangolin.net/self-host/advanced/integration-api)  
* API key with GET/LIST  

#### Usage
1. Import skill files into your workspace/skills area.  
2. Create `PANGOLIN_ADDRESS` & `PANGOLIN_API_KEY` environment variables.
